# LocoReach Analysis Code

Analysis and figure-generation code accompanying the publication:

> **Cerebellar molecular layer interneurons encode locomotor kinematics during
> precise stepping on a rung ladder**
> Michael Graupner et al.

---

## Raw data

Raw electrophysiology, behavioral video, rotary encoder, and synchronization data
for all 11 animals are publicly available on Zenodo:

**https://doi.org/10.5281/zenodo.19708054**

See `tools/README_zenodo.md` for a full description of the data format and structure,
and `tools/README_packaging.md` for how the raw data archives were created.

## Analysis pipeline

### Step 1 — Video pre-processing

1. **`extractBehaviorAndWhiskerCameraTiming.py`** — extracts camera frame timestamps
   from the DAQ trigger signal (`CameraGigEBehavior/daqResult.ma`)
2. **`getRawBehaviorImagesSaveVideo.py`** — exports the behavioral video from the raw
   HDF5 `.ma` format
3. **`extractRungLocation.py`** — detects rung positions in each video frame
4. **DeepLabCut** *(external)* — paw tracking from behavioral video.
   See https://github.com/DeepLabCut/DeepLabCut
5. **`extractPawTrackingOutliers.py`** — removes tracking outliers from DLC trajectories
6. **`extractSwingStancePhases.py`** — detects swing and stance phases from paw positions

### Step 2 — Rotary encoder

- **`getWalkingActivity.py`** — extracts running speed from the rotary encoder signal
  (`RotaryEncoder.ma`)

### Step 3 — Electrophysiology

1. **`tools/generatePsortFiles.py`** — converts raw AxoPatch200 `.ma` files into the
   format expected by PSort
2. **PSort** *(external)* — spike sorting software for loose-seal cell-attached recordings.
   See https://github.com/esedaghatnejad/psort
3. **`analyzePsortTrainClustering.py`** — processes and packages PSort output across
   all animals and recording days
4. **`analyzeEphysPsortData.py`** — per-cell ephys analysis from PSort results

### Step 4 — Combined ephys + behavior

- **`getPsortWalkingActivityAndPawTraces.py`** — aligns spike data with paw trajectories
  and walking speed per trial
- **`getPsortWalkingActivityAndPawTracesCalculatePSTH.py`** — computes peri-event
  time histograms (PSTHs) aligned to swing/stance onset

### Batch processing

- **`runAnalysisOnMultipleMice.py`** — runs any of the above scripts across multiple
  animals; configured via `simplexAnimalEphy.config`

### Step 5 — Group analysis (`groupAnalysisScripts/`)

| Script | Purpose |
|--------|---------|
| `generateAllMouseDictionary.py` | assembles per-animal data dictionaries |
| `PSTHGroupAnalysis*.py` | group-level PSTH statistics (stance/swing onset, step subsets) |
| `ComplexSpikesGroupAnalysis.py` | complex spike analysis |
| `CrossCorrelation*.py` | spike cross-correlations across MLIs |
| `clusterMLIkernelprofiles.py` | t-SNE / clustering of MLI response profiles |
| `clusterPCkernelprofiles.py` | t-SNE / clustering of PC response profiles |
| `generatePSTHGroupAnalysisFigures.py` | intermediate group figures |

## Dependencies

- Python ≥ 3.8
- numpy, scipy, matplotlib
- h5py
- opencv-python
- scikit-learn
- [DeepLabCut](https://github.com/DeepLabCut/DeepLabCut) — paw tracking (Step 1)
- [PSort](https://github.com/esedaghatnejad/psort) — spike sorting (Step 3)
